# FlyCommand
一个允许玩家飞行的mod  
目前只支持版本mc-1.20.1_fabric-0.16.10+  

## 该mod提供飞行命令
- /fly （切换模式）
- /fly on （强制开启）
- /fly off （强制关闭）

## 客户端相关
- 安装mod则飞行中挖掘速度与服务端同步
- 未安装mod则仅秒破方块效果与服务端相同

## mod特性
- 退出服务器、存档后保存mod数据到存档内的 **\"playerdata/\[玩家UUID\].dat\"** 文件
- 开启飞行命令后，切换游戏模式不丢失飞行状态
- 生存、冒险模式下，退出飞行状态的那一次落地，免疫摔落伤害
- 生存、冒险模式下，飞行状态下或刚退出飞行还未落地之前，避免空中挖掘减速惩罚（需要客户端安装mod以获得完整支持）

## 修复的飞行状态丢失问题
- 跨纬度传送丢失飞行状态（未发送客户端更新）
- 末地门回到主世界丢失飞行状态（数据未拷贝）
